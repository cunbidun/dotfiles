# A dotfiles repo

## Dotfiles included: 
1. alacritty
2. Code (vscode)
3. nvim 
4. some script
5. Xresources
6. awesome (my window manager)
7. espanso
8. tmux
9. zsh

## How to install and use the dotfiles:
1. Install [stow](https://www.gnu.org/software/stow/)
2. Clone the repo
3. `cd ~/dotfiles`
4. `stow`
